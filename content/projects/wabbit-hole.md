A Wikipedia rabbit-hole reader: every article you open lands as a card on a stack, the stack becomes a saveable trail, and a daily race game gives the wandering a scoreboard. An LLM-generated trivia layer quizzes you on wherever you ended up.

It’s also the strongest proof on this page of agent engineering on a 2026 stack — spec → implement → review, run almost entirely through an agent workflow, including a 39KB API due-diligence document before the first feature shipped. Vite and vanilla TypeScript up front; Vercel Functions, Neon Postgres with Drizzle, and Clerk behind. Live at wabbithole.io.
