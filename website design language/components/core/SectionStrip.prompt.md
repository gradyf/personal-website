Mono micro-caps label between two full-width rules — the standard way to open a section.

```jsx
<SectionStrip left="SELECTED WORK" right="2024 → NOW" />
<SectionStrip left="WRITING" right="00 POSTS — SOON-ISH" strong={false} />
```

`strong` (default true) uses full-ink rules; set false for softer in-page dividers. Right label is optional.
